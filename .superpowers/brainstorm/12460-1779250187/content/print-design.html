7 处打印需求 · CSS @media print 模板 · print-js 调起浏览器打印

1. 技术方案

B

iframe 打印模板 + print-js（推荐）

用 print-js 调起浏览器打印，每个票据准备独立的打印模板组件

优势

  • CSS 完全控制打印布局
  • 支持打印预览
  • 不污染主页面样式

实现

  • 安装 print-js
  • 每个单据写独立打印模板
A

window.print + CSS @media

直接调用浏览器打印当前页面

优势

  • 零依赖

劣势

  • 打印布局受限
  • 无法隐藏UI控件

4. 打印触发方式

三种触发模式
模式A — 按钮触发：在详情抽屉/弹窗底部放置「🖨️ 打印」按钮，点击立即生成打印模板
模式B — 成功自动弹出：接车开单成功 / 收款成功后，弹出提示"是否打印？"
模式C — 列表批量打印：工单列表用 checkbox 多选后，批量打印选中工单
