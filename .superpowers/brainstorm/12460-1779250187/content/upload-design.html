1. 技术方案

B

multer + 本地磁盘存储（推荐）

后端 multer 接收文件存到 uploads/ 目录，ServeStaticModule 提供静态访问

优势

  • 零外部依赖费用
  • 内网部署无网络延迟
  • 实现简单

注意

  • 需要定期备份 uploads 目录
  • 磁盘空间需监控
A

OSS 云存储

上传到阿里云/腾讯云 OSS

优势

  • 无限扩容

劣势

  • 需要公网/费用
  • 配置复杂

5. 后端 API 设计

文件上传接口
POST /upload/image?module=customers → 单张图片上传 → 返回 { url: "/uploads/customers/xxx.jpg" } POST /upload/images?module=vehicles → 多图上传 → 返回 { urls: ["/uploads/vehicles/xxx.jpg", ...] } POST /upload/file?module=purchase → 附件上传(pdf/doc) → 返回 { url: "/uploads/purchase/xxx.pdf" } GET /uploads/* ← ServeStatic 静态文件访问 限制：图片 max 5MB，附件 max 20MB
